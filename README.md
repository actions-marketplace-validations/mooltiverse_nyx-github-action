Use [Nyx](https://github.com/mooltiverse/nyx) from within GitHub Actions.

[Nyx](https://github.com/mooltiverse/nyx) is a powerful, flexible and extremely configurable semantic release tool. You can put release management on auto pilot regardless of the kind of project, languages, tools and technologies or you can control any aspect of release management manually.

For a reference on Nyx, its capabilities and configuration please jump to the [documentation](https://mooltiverse.github.io/nyx/).

This project is a companion to Nyx as it provides the GitHub Action for ease of use. This guide only focuses on how to use the GitHub Action, while for an overview on Ny please refer to the [main project](https://github.com/mooltiverse/nyx).
